**一款全方位的 WordPress 外掛，專為將 Ti-da 部落格內容順利遷移至 WordPress 而設計。**

## 關於

**TIDA Importer 2026** 是一款全方位的 WordPress 外掛，專為將 Ti-da 部落格內容順利遷移至 WordPress 而設計。它支援 MT (Movable Type) 匯出格式，並具備圖片處理、分類重建及舊連結重新導向等進階功能。

> **請備份資料庫與檔案**
> 在執行匯入之前，請務必備份您的 WordPress 資料庫與檔案。匯入過程具有風險，備份可確保在發生任何問題時能還原您的網站。

---

## 與標準匯入器的差異

與 WordPress 官方的 [Movable Type and TypePad Importer](https://wordpress.org/plugins/movabletype-importer/) 相比，本外掛專為 Ti-da 部落格設計，差異如下：

| 功能 | 標準匯入器 | TIDA Importer 2026 |
| :--- | :---: | :---: |
| **外部圖片** | ❌ (直連) | ✅ **自動下載** |
| **分類階層** | ⚠️ (扁平化) | ✅ **重建親子階層** |
| **舊網址重導向** | ❌ (無) | ✅ **自動建立** |
| **大量匯入** | ⚠️ (易超時) | ✅ **中斷恢復功能** |
| **精選圖片** | ❌ (無) | ✅ **自動設定** |
| **標籤 / 關鍵字** | ⚠️ (Bug) | ✅ **修正** |
| **匯入識別** | ❌ (無) | ✅ **自動標籤** |

### 主要差異詳情
*   **圖片自動下載**: 自動偵測文章內的外部圖片並下載至媒體庫，替換原始連結，避免圖片失效或盜連。
*   **分類結構重建**: 透過分析原始部落格網址，還原完整的分類階層 (父分類與子分類) 關係。
*   **智慧重導向**: 自動建立從舊 Ti-da 網址格式 (如 `e12345.html`) 跳轉到新文章的重導向，避免 SEO 流量流失。
*   **穩定匯入**: 即使數千篇文章，也能利用中斷恢復 (Resume) 功能分批完成，不會因超時而失敗。

## 功能

*   **MT 格式匯入**: 支援 Ti-da 部落格的標準 Movable Type 匯出格式 (`.txt`)。
*   **中斷恢復功能**: 自動儲存進度。若匯入過程因超時或意外關閉而中斷，可從上次停下的地方繼續執行。
*   **圖片處理**:
    *   **自動下載**: 偵測文章內的外部圖片，將其下載至媒體庫並替換內容連結。
    *   **重複防止**: 使用網址雜湊 (Hash) 技術，避免在不同文章中重複下載同一張圖片。
    *   **支援相對協定 URL**: 正確處理 `//example.com/image.jpg` 格式的網址。
    *   **自動精選圖片**: 自動將文章中發現的第一張圖片設為精選圖片 (特色圖片)。
*   **分類重建**:
    *   分析原始部落格網址以重建分類階層 (父分類 > 子分類)。
    *   提供視覺化的「拖放」介面，讓您在其確定前手動調整結構。
*   **舊連結重新導向**: 建立從舊 Ti-da 網址格式 (例如 `e12345.html`) 跳轉至新 WordPress 文章的重新導向。

## 安裝

1.  從 [Release 頁面](https://github.com/ns29qrk/angra-tida-importer-2026/releases) 下載最新的 `tida-importer-2026.zip`。
2.  前往 WordPress 管理後台的 **外掛 > 安裝外掛 > 上傳外掛**。
3.  選擇下載的 zip 檔案並點擊 **立即安裝**。
4.  安裝完成後，點擊 **啟用外掛**。

---

## 使用方法

### 步驟 1: 上傳與匯入
1.  前往管理後台側邊欄的 **TI-DA Importer**。
2.  上傳您的 Ti-da 部落格匯出檔案 (`.txt`)。
3.  (選填) 輸入您的 **部落格網址 (Blog URL)** 以啟用網站地圖分析與分類重建功能。
4.  點擊 **執行匯入 (Run Import)**。
    *   畫面會出現載入遮罩以防止誤觸。
    *   在過程完成前 **請勿關閉分頁**。

### 步驟 2: 分類重建
1.  匯入完成後，點擊 **下一步:分類重建** (或前往 TI-DA Importer > Category Reconstruction)。
2.  確認已輸入 **部落格網址**。
3.  外掛將分析原始網站的分類結構。
4.  在視覺化編輯器中檢視與調整結構:
    *   拖放以重新排序。
    *   使用 `<` (減少縮排) 與 `>` (增加縮排) 按鈕變更階層。
5.  點擊 **建立分類 (Create Categories)** 以將結構套用至匯入的文章。
