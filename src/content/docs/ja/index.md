**Antigravityで昔のプログラムを近代化、てぃーだブログ引越し大作戦**

## 概要

**TIDA Importer 2026** は、てぃーだブログから WordPress へのコンテンツ移行をスムーズに行うための包括的なプラグインです。MT（Movable Type）形式のエクスポートファイルをサポートし、画像の自動取得、カテゴリー階層の再構築、旧URLからのリダイレクト設定など、高度な機能を備えています。

> **データベースとファイルのバックアップについて**
> インポートを実行する前に、必ずWordPressのデータベースとファイルのバックアップを作成してください。インポートにはリスクが伴います。万が一の問題に備え、復元可能な状態にしておくことを強く推奨します。

---

## 開発の背景: 昔作った引っ越しプラグインを近代化した

てぃーだブログには投稿してきた記事をエクスポート(書き出し)する機能がついています。これを利用して、他のブログやCMSに引っ越しすることができます。

Movable Type 形式で出力されるので WordPress では、公式の [Movable Type and TypePad Importer](https://ja.wordpress.org/plugins/movabletype-importer/) を使ってインポート(読み込み)するのが普通の流れです。

ただ、そのファイル内には不足している情報があるので、2015年ごろにその部分を補うWordpress用のインポータをつくりました。

今回は、そのプラグインを Antigravity と WordPress 6 と PHP8で動くように **近代化** することにしました。

近代化ついでに、いくつか仕様を追加しました。WordPressの標準インポーターと比べて以下の違いがあります。

| 機能 | 標準インポーター | TIDA Importer 2026 |
| :--- | :---: | :---: |
| **外部画像** | ❌ (直リンク) | ✅ **自動ダウンロード** |
| **カテゴリー階層** | ⚠️ (フラット化) | ✅ **親子階層を再現** |
| **旧URLリダイレクト** | ❌ (非対応) | ✅ **自動作成** |
| **大規模インポート** | ⚠️ (タイムアウト) | ✅ **中断・再開可能** |
| **アイキャッチ** | ❌ (設定なし) | ✅ **自動設定** |
| **タグ・キーワード** | ⚠️ (バグあり) | ✅ **バグ対応済み** |
| **インポート識別** | ❌ (なし) | ✅ **タグ付与** |

### 主な違いの詳細
*   **画像の自動保存**: 記事内の外部画像を自動的にメディアライブラリへ保存し、リンクを書き換えます（直リンクを防ぎます）。
*   **カテゴリの完全再現**: 元のブログURL構造を解析し、失われがちな親子関係を正しく再構築します。
*   **スマートなリダイレクト**: `e12345.html` のようなてぃーだブログ特有のURLから、新しい記事URLへの転送設定を自動で行います。
*   **安定したインポート**: 記事数が多い場合でも、進捗を保存しながら確実に処理を進められます（レジューム機能）。

## 機能詳細

*   **MT形式インポート**: てぃーだブログからエクスポートされた標準的な形式（`.txt`）に対応しています。
*   **レジューム（中断再開）機能**: 進捗を自動保存します。タイムアウトや誤操作で中断しても、続きからインポートを再開できます。
*   **画像処理**:
    *   **自動ダウンロード**: 記事内の外部画像を検出し、メディアライブラリにダウンロードしてリンクを置換します。
    *   **重複防止**: URLハッシュを使用して、複数の記事で同じ画像が使われている場合に重複ダウンロードを防ぎます。
    *   **プロトコル相対URL対応**: `//example.com/image.jpg` 形式のURLも正しく処理します。
    *   **アイキャッチ自動設定**: 記事内で最初に見つかった画像を自動的にアイキャッチ（サムネイル）として設定します。
*   **カテゴリー再構築**:
    *   元のブログURLを解析し、カテゴリーの親子階層構造を復元します。
    *   視覚的な「ドラッグ＆ドロップ」インターフェースで、確定前に構造を自由に調整できます。
*   **レガシーURLリダイレクト**: 古いてぃーだブログのURL形式（例: `e12345.html`）から、新しい WordPress 記事へのリダイレクトを作成します。

## てぃーだブログでの設定

### エクスポートファイルの取得方法
1. 「管理画面」の「サイドバー」にある、「ブログの設定」の「書き出し」を選択してください.  
![てぃーだブログ|書き出しメニュー](/tida-importer-2026/images/common/note-2025122901.webp)

2. 書き出しの設定
    - 文字コード: UTF-8 を選んでください
    - 対象年: すべて選択 (デフォルト)
    - 対象カテゴリ: すべてを選択
    - 設定が終わったら、「書き出し開始」をクリックしてください
    
    ![てぃーだブログ|書き出し項目](/tida-importer-2026/images/common/note-2025122902.webp)

3. 保存先を選択して、「保存」をクリックしてください.  
![てぃーだブログ|書き出し保存先](/tida-importer-2026/images/common/note-2025122903.webp)

### サイドバーにカテゴリを表示

カテゴリの階層構造を取得するために、サイドバーのカテゴリープラグインを表示するように設定してください
1. 「管理画面」の「サイドバー」にある、「ブログの設定」の「サイドバー」を選択してください.  
![てぃーだブログ|カテゴリメニュー](/tida-importer-2026/images/common/note-2025122904.webp)

2. カテゴリープラグインを表示するように設定してください.  
![てぃーだブログ|書き出しメニュー](/tida-importer-2026/images/common/note-2025122905.webp)   
より詳しい説明は[公式](https://faq.ti-da.net/e408411.html)を参照してください

## インストール方法

1.  [Releases ページ](https://github.com/ns29qrk/angra-tida-importer-2026/releases) から最新の `tida-importer-2026.zip` をダウンロードします。
2.  WordPress 管理画面の **プラグイン > 新規プラグインを追加 > プラグインのアップロード** に移動します。
3.  ダウンロードした zip ファイルを選択し、**今すぐインストール** をクリックします。
4.  インストール完了後、**プラグインを有効化** します。

## TIDA Importer 2026の使用方法

1. サイドバーから 'TI-DA Importer' を選択
![管理画面メニュー](/tida-importer-2026/images/common/note-2025122911.webp)

2. アーカイブのアップロード
    *   ①で、てぃーだブログからエクスポートしたファイルを選択してください
    *   ②に、てぃーだブログで使っていたURLを入力してください。入力しなくてもインポートは出来ますが、入力した場合はリダイレクトやカテゴリの親子関係、画像の取得が可能となります
    *   設定が終わったら③の「アップロードして解析」ボタンをクリックしてください
![アップロード画面](/tida-importer-2026/images/common/note-2025122912.webp)

3. アップロードしたファイルの解析が行われます.  
    解析が終わったら 'インポート実行' ボタンをクリック
![解析画面](/tida-importer-2026/images/common/note-2025122913.webp)

4. インポート中です。データ取得に時間がかかる場合があります（タブを閉じないでください）
![インポート中](/tida-importer-2026/images/common/note-2025122915.webp)

5. インポート完了です.   
    'カテゴリーの再構築' ボタンを押すとカテゴリーの親子関係の調整ができます.    
    (ブログURLを指定していない場合はここで終了です)
![インポート完了](/tida-importer-2026/images/common/note-2025122916.webp)

6. カテゴリー再構築画面
![カテゴリ再構築](/tida-importer-2026/images/common/note-2025122917.webp)
  *   '>' をクリックすると子カテゴリーになります
    ![子カテゴリ化](/tida-importer-2026/images/common/note-2025122919.webp)
  *   '<' をクリックすると親カテゴリーになります
    ![親カテゴリ化](/tida-importer-2026/images/common/note-2025122920.webp)
  *   ドラッグでカテゴリーの位置を変更できます
    ![ドラッグ移動](/tida-importer-2026/images/common/note-2025122921.webp)
  *   'カテゴリを更新/作成' ボタンをクリックすると適用されます

7. カテゴリの再構築が完了すればインポート終了です。
  ![完了画面](/tida-importer-2026/images/common/note-2025122922.webp)
